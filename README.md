# Portfolio-site

My personal portfolio site built with [Next.js](https://nextjs.org), configured with Docker, API rate limiting, and Resend email integration.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
